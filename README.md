# front-end-teste
Teste desenvolvido para contratatação de front-end.
Orientações
Pelo cmd localize a pasta com seu nome no desktop e digite:
npm run dev

Sua aplicação estará rodando em: http://localhost:4000/
Os arquivos da sua aplicação estarão na pasta: public/
Depois do seu setup você precisará seguir as seguintes instruções, para construir a aplicação:
1.	Desenvolver a página, que tem o seu layout em arte/layout.psd.(A única fonte usada foi a Open Sans, disponível no Google Fonts)
2.	Na galeria de imagens você pode usar a biblioteca js. que desejar;
3.	O uso da frameworks e pré processadores css é livre
4.	Pesquise a vontade.

Requisitos não obrigatórios
Criar um branch com seu nome e envie para o git.
